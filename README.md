# Ecommerce SQL Analytics Project

This project contains a PostgreSQL ecommerce database schema and a collection of SQL analytics queries.

## Features

- Ecommerce database schema
- Customer analytics queries
- Revenue and sales analysis
- Product profitability analysis
- Return and refund analysis
- Cohort and RFM analysis
- Advanced SQL techniques using:
  - JOINs
  - CTEs
  - Window Functions
  - Aggregations
  - Ranking Functions

## Main Tables

- customers
- orders
- products
- order_items
- returns

| File | Description |
